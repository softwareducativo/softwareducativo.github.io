Para utilizar as teclas de atalho, existem comandos específicos de cada navegador:
  • Ir para o Conteúdo
    Google Chrome: Alt + 1
    Mozilla Firefox: Alt + Shift + 1
  • Ir para o menu
    Google Chrome: Alt + 2
    Mozilla Firefox: Alt + Shift + 2
  • Ir para o Rodapé
    Google Chrome: Alt + 4
    Mozilla Firefox: Alt + Shift + 4
  • Ir para o Menu de Acessibilidade
    Google Chrome: Alt + 5
    Mozilla Firefox: Alt + Shift + 5
  • Modo alto contraste
    Google Chrome: Alt + 8
    Mozilla Firefox: Alt + Shift + 8
